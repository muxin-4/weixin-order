# 日志
- 一套能实现日志输出的工具包
- 能够描述系统运行状态的所有时间都可以算作日志

应用场景
- 用户下线
- 接口超时
- 数据库崩溃
- Hello World


日志框架的能力
- 定制输出目标
- 定制输出格式
- 携带上下文信息
- 运行时选择性输出
- 灵活的配置
- 优异的性能

- 日志门面

- 日志实现
